# Dis-Via
Dis-Via is a web app that use the Keolis API to get times for tramway and bus in Dijon.

It's webapp capable, and can be add to home screen on Android to make it look like a native app.
It use the localStorage to save your preferences, so the settings should stay much longer than with cookies or sessions vars.

If you wish to use this application, you can visit the online version at http://dev.apcros.fr/Dis-Via
if you want to host it yourself and make it public, please contact me before.
